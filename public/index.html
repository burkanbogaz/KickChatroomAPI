API Endpoints:

  • GET /api/info - Get server status and active channels
  • GET /api/messages/:channelName - Get all messages for a channel
  • POST /api/monitor - Start monitoring a channel (body: {channelName: "name"})
  • POST /api/unmonitor - Stop monitoring a channel (body: {channelName: "name"})
